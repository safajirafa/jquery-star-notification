Star notification jQuery plugin

An iOS like toast notification

Download this project as a .zip file Download this project as a tar.gz file

An iOS like notification toast to star and unstar items (webkit browsers)

Demo

This plugin works as a decoupled widget with the following features:

  • Uses PUB/SUB pattern
  • No dependencies on images
  • Easy integration
  • CSS3 based animation (for webkit browsers)

Plugin initialization

var options = {
    fadeOutTime: 800, // Same time as animation duration in css
    starChar: '&#9733;',
    unstarChar: '&#9734;',
    starLabel: 'Starred',
    unstarLabel: 'Unstarred' 
};
// Parameters are optional
$(document).starNotification(options);

Publish an event - notify starred item

$(document).trigger('starNotification', 'starred');

Notify unstarred item

$(document).trigger('starNotification', 'unstarred');
